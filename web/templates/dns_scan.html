Reconnaissance DNS Complète

Collecte d'informations DNS avancée et reverse lookup

ℹ️ Fonctionnalités DNS

  • DNS Lookup : Recherche tous les enregistrements DNS (A, AAAA, MX, NS, TXT, SOA, CNAME)
  • Reverse DNS : Trouve le nom de domaine associé à une IP
  • Multiple IPs : Liste toutes les IPs associées à un domaine
  • WHOIS : Informations sur le propriétaire du domaine (Linux/Unix seulement)
Entrez un nom de domaine (sans http:// ou www si non nécessaire)

📝 Exemples de domaines

google.com github.com cloudflare.com

Entrez une adresse IPv4 pour trouver le domaine associé

📝 Exemples d'adresses IP

8.8.8.8 (Google DNS) 1.1.1.1 (Cloudflare DNS) 208.67.222.222 (OpenDNS)
