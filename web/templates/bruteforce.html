Brute Force SSH • FTP • HTTP

Testez la robustesse des identifiants d'un service

⚠️ Avertissement

N'utilisez *jamais* cet outil sur une cible sans autorisation légale explicite.

ℹ️ Instructions

  • Entrez l'IP ou domaine cible (ex: 192.168.1.10)
  • Choisissez le service : SSH / FTP / HTTP
  • Indiquez l'utilisateur à tester
  • Saisissez une wordlist (un mot de passe par ligne) OU chargez depuis une URL
Entrez l'URL d'une wordlist (téléchargée côté serveur).
Exemple: https://raw.githubusercontent.com/danielmiessler/SecLists/refs/heads/master/Passwords/Common-Credentials/100k-most-used-passwords-NCSC.txt
